Изучай Манипуляторы

Декартовый манипулятор с вакуумным захватом управляется командами перемещения по осям X, Y, Z.

Система команд Python
КомандаОписаниеПример
connect(ip, port)Подключение к роботуconnect("192.168.1.10", 6000)
moveposition(x, y, z)Линейное перемещение (X→Y→Z)moveposition(5, 3, 0)
setposition(x, y, z)Быстрый переход (без анимации)setposition(0, 0, 10)
vacuum(state)Вакуумный захват (True/False)vacuum(True)
Правила физики

Покомпонентное движение

moveposition двигает строго по осям: сначала X, затем Y, затем Z. Диагональные движения запрещены.

Вакуумный захват

Захват берет верхний кубик в точке. Отпускает кубик на верх доступного пространства (другого кубика или пол Z=0).

Стекинг (Горка)

Кубики ставятся друг на друга. Чтобы взять нижний, нужно сначала убрать верхний.

Безопасность

Поднимайте Z перед горизонтальным перемещением, чтобы не столкнуться с кубиками.
